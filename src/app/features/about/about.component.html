About This Project

This personal accounting application is a comprehensive financial management tool designed to help users track income, expenses, capital, and purchases.

Key Features

  • Income and Expense Tracking: Accurately record income and expenses for detailed financial analysis.
  • Budgeting: Create and manage budgets to control spending and achieve financial goals.
  • Capital and Purchase Tracking: Monitor capital investments and purchases for asset management.
  • Financial Reporting: Generate customizable reports on net income, assets, and other key financial metrics.

Highlights

  • User-Friendly Interface: Prioritized a clean and intuitive design for easy navigation and data entry.
  • Robust Features: Comprehensive functionality for effective financial management.
  • Scalability: Designed to handle growing financial needs and data volumes.

Technologies

  • Angular 18
  • Angular Material
  • RxJs
  • NgRx
  • Firebase
  • Sass

GitHub Repository: https://github.com/moof44/personal-accounting-2

This application is a valuable resource for individuals seeking to improve their financial organization and make informed decisions.
